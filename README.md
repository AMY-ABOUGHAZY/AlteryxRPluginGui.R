# AlteryxRPluginGui.R

Reskin R Tool

## Installation

1. Download the [zip file](https://github.com/alteryx/AlteryxRPluginGui.R/archive/master.zip)
2. Unzip the files.
3. Rename `AlteryxPluginGui.R-master` to `AlteryxPluginGui`
4. Move it to `C:\Program Files\Alteryx\bin\HtmlPlugins`
5. Restart Alteryx.
